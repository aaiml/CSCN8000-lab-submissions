from sklearn.model_selection import train_test_split
from sklearn.naive_bayes import GaussianNB, MultinomialNB
from sklearn.metrics import accuracy_score, confusion_matrix
import matplotlib.pyplot as plt
import seaborn as sns

features = bag_of_words_df.drop('Spam', axis=1)
target = bag_of_words_df['Spam']
X_train, X_test, y_train, y_test = train_test_split(features, target, test_size=0.2, random_state=42)
In [14]:
gnb = GaussianNB()
gnb_predictions = gnb.fit(X_train, y_train).predict(X_test)

The Multinomial Naive Bayes model outperformed the Gaussian Naive Bayes model in terms of accuracy. It achieved an accuracy of 0.9787, which is higher than the accuracy of the Gaussian Naive Bayes model, which was 0.9546. This indicates that the Multinomial Naive Bayes model made more accurate predictions overall. Additionally, when evaluating the Multinomial Naive Bayes model using precision, recall, and F1-score, it showed consistently high performance across both classes (0 and 1). The precision and recall values for both classes were relatively high, indicating a good balance between correctly identified positive instances and overall positive instances captured. This suggests that the Multinomial Naive Bayes model is more effective in correctly classifying instances and has better overall performance compared to the Gaussian Naive Bayes model.
